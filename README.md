# Emoji Query

## Install

```js
npm install emoji-query
```

## Usage

```js
$ emoji-query

#️⃣   :hash:                        #number sign + combining enclosing keycap

$ emoji-query dog

🐶   :dog:                         #dog face
🐕   :dog2:                        #dog
🐩   :poodle:                      #poodle
```
